# FWCB Lazy CWL Directory

A premium and dynamic clan directory website built for the Clash of Clans (CoC) community. This platform is specifically designed to manage the "Lazy CWL" event, allowing players to easily find and join the right clan based on their Town Hall requirements.

---

### Key Features
* **Dynamic Admin Panel:** Add, edit, update, and delete clan information on the fly without touching the codebase.
* **Custom Section Reordering:** Sort and manage the display order of Town Hall sections directly from the admin dashboard.
* **Dark Gaming UI:** A professional dark theme featuring glowing hover effects, clean typography, and smooth CSS animations.
* **Fully Responsive:** 100% optimized for seamless viewing across mobile, tablet, and desktop devices.
* **Real-Time Backend:** Powered by Google Firebase for instant database updates and secure administrator authentication.

### Technology Stack
* **Frontend:** HTML5, CSS3 (CSS Grid & Flexbox), Vanilla JavaScript (ES6)
* **Backend & Database:** Google Firebase (Cloud Firestore)
* **Authentication:** Firebase Auth (Email/Password)
* **Deployment:** GitHub Pages

---

### Setup Instructions

1. **Clone the Repository:** Download or clone this repository to your local machine.
2. **Configure Firebase:** Open `index.html` and `admin.html` and replace the placeholder Firebase configuration with your own API keys.
3. **Add Assets:** Ensure the following images are placed in the root directory:
   * `logo.png` (Your main community logo)
   * `th11.png` through `th18.png` (Transparent Town Hall images)
4. **Deploy:** Push the files to your GitHub repository and enable **GitHub Pages** from the repository settings to make the site live.
